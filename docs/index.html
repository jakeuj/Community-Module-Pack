為繁中玩家整理

不只翻譯按鈕，連新活動也補齊

針對目前事件資料逐筆比對，避免新增內容找不到資源時退回英文。

01

完整繁中名稱

世界首領、大型活動、團隊活動、匯流與冒險分類皆提供繁中名稱。

02

即時活動提醒

追蹤想參加的活動，在開始前收到通知，也能快速複製附近傳送點。

03

自動建置發布

每個版本都由 GitHub Actions 重新建置、檢查翻譯覆蓋並封裝成 BHM。

四步完成

安裝方式

  1. 1
    關閉 Blish HUD

    替換或新增模組前，先完全結束 Blish HUD。

  2. 2
    下載 BHM

    從最新 Release 下載 Events.Module.bhm。

  3. 3
    放進 Modules

    將檔案放到 Documents\Guild Wars 2\addons\blishhud\Modules。

  4. 4
    重新啟動並啟用

    在 Blish HUD 的模組設定中找到此模組並啟用。

若文件資料夾由 OneDrive 同步，實際路徑可能位於 OneDrive。可參考 Blish HUD 官方模組安裝說明。
