fork.5 新功能

手動更新這一次，
之後穩定版自動跟上。

模組每次載入時會在背景檢查 GitHub 最新穩定版。下載通過 SHA-256 驗證後，會保留模組啟用狀態並重新啟動 Blish HUD；任何檢查、下載或驗證失敗都繼續使用原版本。

  • 01
    預設開啟

    自動更新可隨時關閉；關閉後仍會通知新版，也能按「立即更新」。

  • 02
    只跟穩定版

    測試版、預發行版、格式錯誤、較舊版本與不受信任的下載網址都不會安裝。

  • 03
    驗證成功才重啟

    GitHub Release 資產與 SHA-256 必須一致，失敗時不替換檔案也不要求重啟。

第一次仍需手動安裝 fork.5

舊版尚未內建更新器，請先依下方步驟安裝一次；從 fork.5 開始，後續合規穩定版才會自動更新。

為繁中玩家整理

看得懂時間，也對得上英文攻略

繁中名稱、英文原名與獎勵摘要集中在同一張事件卡片；活動時程仍以官方 Wiki Widget 為準，不需要依賴另一份第三方計時表。

01

世界王獎勵資訊

13 隻核心世界王顯示保底稀有／特異裝備與地面寶箱龍晶礦；滑鼠提示說明帳號／角色每日限制、Wiki 來源與查核日期。

02

繁中＋英文對照

事件卡片以繁中名稱為主、官方英文原名為輔；搜尋同時支援繁中、英文與既有別名，查英文攻略不用來回猜名稱。

03

官方時程與離線備援

104 個事件的時間、地圖、傳送點與 Wiki 連結跟隨官方資料；連線失敗時依序使用成功快取與內建資料，通知與 11 張專屬圖示仍可使用。

fork.5 首次安裝 · 四步完成

安裝方式

  1. 1
    關閉 Blish HUD

    替換或新增模組前，先完全結束 Blish HUD。

  2. 2
    下載 BHM

    從最新 Release 下載 Events.Module.bhm。

  3. 3
    放進 Modules

    將檔案放到 Documents\Guild Wars 2\addons\blishhud\Modules。

  4. 4
    重新啟動並啟用

    在模組設定中啟用；「自動更新此模組」預設開啟。

若文件資料夾由 OneDrive 同步，實際路徑可能位於 OneDrive。可參考 Blish HUD 官方模組安裝說明。
